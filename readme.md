1. NeoPixel-Bibliothek in Thonny installieren
- MicroPython hat die NeoPixel-Unterstützung oft schon eingebaut. Falls nicht, installieren wir sie kurz:
- Klicke in Thonny oben im Menü auf Tools -> Manage Packages (oder Werkzeuge -> Pakete verwalten).
- Suche nach neopixel. Klicke auf das gefundene Paket und installiere es.
